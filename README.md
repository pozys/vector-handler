# Обработчик векторов

Производит манипуляции над векторами

## Требования

- PHP >=7.0

## Установка

```bash
composer require pozys/vector-handler
```

## Использование

```php
$counter = new NumberCounter();
$numbers = [555, 901, 482, 1771];
echo $counter->countWithEvenDigitNumber($numbers); // 1
```
